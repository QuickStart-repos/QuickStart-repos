# QuickStart-repos

This repository provides community tools used across the organization, including issue templates, discussion templates, and contribution guidelines.

## Available Repositories

Here are the current project templates maintained by the organization:

- [GitHub Static Website Template](https://github.com/QuickStart-repos/GitHub-static-website-template)

More templates will be added over time. If you'd like to request a new one, feel free to open an issue using the request form.

## Contributing

If you notice a problem or want to improve something in this repository, please follow the contribution guidelines in [`CONTRIBUTING.md`](https://github.com/QuickStart-repos/.github/new/main?tab=CONTRIBUTING.md).
All contributions are welcome through forks and pull requests.
